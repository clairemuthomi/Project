# Layali Café & Desserts Website

## About the Project

This project is a responsive website developed for **Layali Café & Desserts**, a local café located in Magharibi, Kenya. The website provides information about the café, its products, and services while offering an attractive and user-friendly experience.

## Features

* Responsive design
* Navigation menu
* Product menu
* Image gallery
* Customer testimonials
* Frequently Asked Questions (FAQ)
* Contact form with validation

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Git
* GitHub

## Website Pages

* Home
* About Us
* Menu
* Gallery
* Testimonials
* FAQ
* Contact Us
